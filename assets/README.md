# Crackd Brand Assets

This directory contains the official brand assets for Crackd.

## Logo Versions

- **[crackd-logo-color.svg](./crackd-logo-color.svg)**: The primary logo. Light text with magenta accent. Best for dark backgrounds.
- **[crackd-logo-dark.svg](./crackd-logo-dark.svg)**: Dark version of the logo. Best for light backgrounds.
- **[crackd-mark-magenta.svg](./crackd-mark-magenta.svg)**: The standalone Cipher Tile mark in brand magenta.
- **[logo-banner.svg](./logo-banner.svg)**: Full brand banner with tagline and dark background.

## Specs

- **Brand Magenta**: `#FF00A8`
- **Bone (Light Text)**: `#EDE6F0`
- **Ink (Dark Text)**: `#040008`
- **Font**: System UI (Semibold / 600 weight)
